# Экспертная система подбора индивидуальных тренировок

Прототип продукционной экспертной системы (ЭС) классифицирующего типа, разработанный в рамках лабораторных работ №3-4 по дисциплине **«Введение в искусственный интеллект»** (УУНиТ, специальность «Программная инженерия»).

Программа анализирует параметры пользователя (цель, инвентарь, уровень подготовки, целевая зона, доступное время) и на основе продукционной базы знаний подбирает оптимальный комплекс упражнений с мультимедийным сопровождением.

## 🛠️ Стек технологий
* **Язык программирования:** C#
* **Платформа:** .NET / Windows Forms
* **База знаний:** JSON (`tree.json`)

## 🧠 Архитектура и модель данных

Система использует **прямой логический вывод** на основе иерархического дерева решений. Граф переходов полностью динамический и загружается из внешнего файла `tree.json`.

Каждый шаг (узел) системы обрабатывается через класс `Node`:

```csharp
namespace VVII_laba3
{
    internal class Node
    {
        public string Id { get; set; }                         // Уникальный идентификатор узла
        public string Question { get; set; }                   // Текст вопроса для текущего шага
        public string Path { get; set; }                       // Путь к изображению для финального результата
        public string Title { get; set; }                      // Текст финальной рекомендации (результат вывода)
        public Dictionary<string, string> Answer { get; set; } // Варианты ответов и ID следующих узлов
        public bool IsFinal { get; set; }                      // Флаг терминального (конечного) узла
    }
}
```
Принцип работы интерфейса:
Инференс-движок считывает tree.json при старте приложения и строит словарь Dictionary<string, Node>.

Интерфейс на Windows Forms динамически генерирует элементы управления (кнопки или варианты ответов) на основе ключей из словаря Answer текущего узла.

При достижении терминального узла (IsFinal = true):

На экран выводится итоговый план тренировки (считывается из поля Title).

В UI динамически подгружается графическая схема или картинка упражнения по пути, указанному в поле Path.


⚙️ Инструкция по запуску и модификации
Запуск приложения
Скомпилируйте проект через Visual Studio (или запустите готовый .exe файл из папки bin/Debug/ или bin/Release/).

Важно: Исполняемый файл приложения жестко привязан к базе знаний. Файл tree.json должен находиться в той же папке, где лежит запущенный .exe, иначе система не сможет инициализировать дерево.

Изменение базы знаний
Вся логика экспертной системы вынесена за пределы кода. Если вам необходимо добавить новые правила, условия или изменить результат:

Откройте файл tree.json в любом текстовом редакторе.

Отредактируйте или добавьте ветку переходов в словарях Answer.

Перезапустите приложение. Изменения вступят в силу автоматически без перекомпиляции C# кода.
